# Birthday Age Calculator (Android)

Android application for tracking birthdays and calculating ages with validation.

## Features
- **Date Picker**: Select birth dates easily
- **Age Calculation**: Precise age computation
- **Data Transfer**: Pass information between activities
- **Input Validation**: Ensure correct date formats
- **Age Restriction**: 18+ year validation
- **Material Design**: Modern Android UI

## Technology Stack
- Kotlin
- Android SDK
- Material Design Components
- Activity navigation

## Use Cases
- Birthday tracking
- Age verification
- Personal information apps
- Date calculations
